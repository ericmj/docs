Modules summary ↑

EEx

EEx stands for Embedded Elixir. It allows you to embed Elixir code inside a string in a robust way:
EEx.AssignsEngine

An abstract engine that, when used with the TransformerEngine, allows a developer to access assigns using @ as syntax
EEx.Engine

This is the basic EEx engine that ships with Elixir. An engine needs to implement three functions:
EEx.SmartEngine

An engine meant for end-user usage that includes EEx.AssignsEngine and other conveniences. Read EEx.AssignsEngine for examples
EEx.TransformerEngine

An abstract engine that is meant to be used and built upon in other modules. This engine implements the EEx.Engine behaviour and provides a transform overridable directive that allows a developer to customize the expression returned by the engine
