EEx.TransformerEngine

An abstract engine that is meant to be used and built upon in other modules. This engine implements the EEx.Engine behaviour and provides a transform overridable directive that allows a developer to customize the expression returned by the engine.

Check EEx.AssignsEngine and EEx.SmartEngine for examples of using this module.
