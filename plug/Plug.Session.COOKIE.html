Plug.Session.COOKIE

  • Summary
  • Functions

Stores the session in a cookie.

Implements a cookie store. This cookie store is based on Plug.MessageVerifier which signs each cookie to ensure they won't be tampered with.

Notice the cookie contents are still visible and therefore private data should never be put into such store.

Options

  • :secret - secret token used to sign the cookie;

Examples

# Use the session plug with the table name
plug Plug.Session, store: :cookie, key: "_my_app_session", secret: "a4f8d34f9"
